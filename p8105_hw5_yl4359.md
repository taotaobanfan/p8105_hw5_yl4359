# Problem 1

``` r
iris_with_missing = iris %>% 
  map_df(~replace(.x, sample(1:150, 20), NA)) %>%
  mutate(Species = as.character(Species))
```

## Write a function

``` r
replace_na = function(vec){
  new_vec = c()
  if (is.numeric(vec)) {
    for (x in vec) {
      if (is.na(x)) {
        x = mean(vec, na.rm = TRUE)
      }
      new_vec = c(new_vec, x)
    }
  }
  if (is.character(vec)) {
    for (x in vec) {
      if (is.na(x)) {
        x = "virginica"
      }
      new_vec = c(new_vec, x)
    }
  }
  new_vec
}
```

## Apply the function

``` r
iris_without_missing = map_dfr(iris_with_missing, replace_na)
```
